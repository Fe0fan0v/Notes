# Приложение онлайн-заметок
Создан на Flask

# Зависимости:

Для установки зависимостей выполните в терминале:<br />

`pip install -r requirements.txt`

# Использование:

Запустите эту команду в терминале:

`python manage.py`

Зайдите на `127.0.0.1:5000` в вашем браузере
